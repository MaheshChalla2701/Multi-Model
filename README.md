# Medical Imaging Analysis Platform

A comprehensive medical imaging analysis platform designed to process X-ray, MRI, and CT scans. The system transforms raw medical images and patient context into structured, actionable medical facts using AI, routing cases to specialist pipelines and generating differential diagnoses based on historical evidence.

## Architecture

The platform consists of several layers:
1. **Medical Understanding Layer (Phase 1 MVP)**: Powered by Gemini 2.5 Pro, this layer acts as the primary intelligence gateway. It performs:
   - Quality Assessment
   - Modality Classification (X-ray, MRI, CT)
   - Anatomy Classification
   - Visual Findings Extraction
   - Retrieval Query Generation
2. **Domain Router** *(Upcoming)*: Routes the structured findings to specialist pipelines.
3. **Specialist Pipelines** *(Upcoming)*: Domain-specific analysis for Chest, Neuro, Bone, etc.
4. **Evidence Collection Layer (Qdrant)** *(Upcoming)*: Retrieves similar cases and medical literature.
5. **Knowledge Graph Layer (Neo4j)** *(Upcoming)*: Maps medical relationships (e.g., Finding -> Disease).
6. **Disease Inference Engine** *(Upcoming)*: Generates differential diagnoses and confidence scores.

## Phase 1 MVP Details

Phase 1 implements the **Medical Understanding Layer**. 

### Tech Stack
* **Frontend**: Next.js with Tailwind CSS, Framer Motion, and Lucide Icons.
* **Backend**: Python FastAPI.
* **AI Engine**: Google Gemini 2.5 Pro (via `google-generativeai` SDK).

### Getting Started

#### Prerequisites
* Node.js (for Next.js)
* Python 3.10+
* A Google Gemini API Key

#### 1. Start the Backend (FastAPI)
```bash
cd backend
python -m venv venv
# Activate virtual environment
# Windows:
.\venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

pip install -r requirements.txt

# Create a .env file and add your API key:
# GEMINI_API_KEY=your_key_here

# Run the server
uvicorn app.main:app --reload
```

#### 2. Start the Frontend (Next.js)
```bash
cd frontend
npm install
npm run dev
```

#### 3. Access the Application
Open your browser and navigate to `http://localhost:3000`. Upload a sample medical scan and provide optional patient context to see the structured output generated by Gemini 2.5 Pro.
